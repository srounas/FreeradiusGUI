Export this server's RADIUS clients and/or certificates to a file, then import it on another FreeRADIUS GUI instance to give it an identical configuration. Admin login and host-specific settings (bind address, config paths) are never included - each server keeps its own.

Importing does not restart FreeRADIUS by itself - go to the Dashboard and click Apply afterward to activate whatever you import here.

Everything imported here is validated the same way as a direct upload on the Certificates page - a certificate/key that don't match each other, or a malformed client list, will be rejected with a specific reason rather than silently applied.
